# Roadmap de Desarrollo (Sprints)

- **Sprint 1 (Semanas 1-3):** Base estructural, autenticación y CRUD de vehículos/gastos.
- **Sprint 2 (Semanas 4-6):** Motor de cálculo (depreciación) y lógica de alertas.
- **Sprint 3 (Semanas 7-9):** Integración de servicios externos (Push, Mail, SMS) y configuración de usuario.
- **Sprint 4 (Semanas 10-12):** QA, seguridad, optimización y preparación de entorno de producción.
